Tippy Inspector

Unity Scene Editor for Banter VR

1

Set up Unity Scene (Optional)

In your Unity scene, create two required GameObjects in the root of your hierarchy:

  1. "Scene" - Any GameObject that is a child of this will be crawled and loaded in Tippy
  2. "People" - This should remain empty (used for player management)

üí° Tip: These GameObjects should exist at the root level of your Unity scene hierarchy for proper functionality.

‚ö†Ô∏è Important: For GameObjects to be crawlable by Tippy, they must have both a BanterTransform and BanterObjectId component attached to them.

Example Unity hierarchy showing Scene and People GameObjects

2

Create a Firebase project

Head to the Firebase Console and create a new project (or use an existing one).

3

Enable Realtime Database

In your Firebase project:

  1. Go to Build ‚Üí Realtime Database
  2. Click Create Database
  3. Choose your database location
  4. Start in test mode (we'll configure rules next)

‚ö†Ô∏è Important: Copy your database URL (e.g., https://your-project-id-default-rtdb.firebaseio.com). You'll need this in step 6.

4

Configure Database Rules

Go to the Rules tab in Realtime Database and paste these rules:

{
  "rules": {
    ".read":  "now < 2056357200000",
    ".write": "now < 2056357200000"
  }
}

Click Publish to save the rules.

5

Enable Anonymous Authentication

Tippy uses Firebase Authentication for secure access:

  1. Go to Build ‚Üí Authentication
  2. Click Get Started
  3. Go to the Sign-in method tab
  4. Enable Anonymous authentication
6

Get your Firebase config

Go to Project Settings ‚Üí General ‚Üí Your apps ‚Üí SDK setup and configuration and copy your config object:

const firebaseConfig = {
  apiKey: "your-api-key",
  authDomain: "your-project.firebaseapp.com",
  projectId: "your-project-id",
  storageBucket: "your-project.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abc123def456",
  databaseURL: "https://your-project-id-default-rtdb.firebaseio.com"
};

Note: Add the databaseURL field with your database URL from step 3.

7

Open your index.html

Navigate to your Banterworld's index.html file.

8

Paste this code

Add the following to your <body> tag, replacing the firebaseConfig with yours:

<body>
    <div id="tippyContainer"></div>

    <script type="module">
        let importUrl = 'https://app.tippy.dev'
        const { TippyInjector } = await import(`${importUrl}/tippy-injector.js`);

        // Initialize Tippy with custom configuration
        const tippy = new TippyInjector({
            targetElement: document.getElementById('tippyContainer'),
            config: {
                serverUrl: importUrl,
                firebaseConfig:{
                    apiKey: "your-api-key",
                    authDomain: "your-project.firebaseapp.com",
                    projectId: "your-project-id",
                    storageBucket: "your-project.appspot.com",
                    messagingSenderId: "123456789",
                    appId: "1:123456789:web:abc123def456",
                    databaseURL: "https://your-project-id-default-rtdb.firebaseio.com"
                }
            }
        });

        // Inject Tippy into the page
        tippy.inject();
    </script>
</body>
9

That's it!

Load your Banterworld and Tippy will be ready to use.
